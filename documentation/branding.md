# Branding

## Color Palette

- Dark Goldenrod: `#A17E0D` (`161, 126, 13`)
- Khaki: `#F5D198` (`245, 209, 152`)
- Dark Khaki: `#CAA369` (`202, 163, 105`)
- Black: `#282725` (`40, 39, 37`)
- Dark Olive Green: `#63573F` (`99, 87, 63`)

## Expert Assessment

This is a strong warm metallic palette with clear brand character. It works especially well for identity assets (logo, icons, hero graphics) and creates a premium bronze/gold look.

## Strengths

- Cohesive color family with intentional tone.
- Strong dark anchor (`#282725`) for text and outlines.
- Clear accent hierarchy between primary bronze (`#A17E0D`) and supporting tones.

## Risks

- Midtones (`#CAA369`, `#63573F`) are close in value and can blend in small UI components.
- Gold tones used for body text may fail accessibility contrast on light backgrounds.

## Recommended Usage

1. Use `#282725` for primary text, icon strokes, and high-contrast UI elements.
2. Use `#A17E0D` as the primary brand accent.
3. Use `#CAA369` for secondary accents and larger surfaces.
4. Use `#F5D198` for highlights and light decorative backgrounds.
5. Reserve `#63573F` for subtle depth or tertiary accents.

## Accessibility Note

Always verify text/background contrast for UI and documentation. Prefer dark text (`#282725`) on light surfaces for readability.
