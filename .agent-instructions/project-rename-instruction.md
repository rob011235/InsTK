# Project Rename Skill

## Description

Rename projects, update namespaces, and adjust references.

## Steps

1. Locate the project file (`.csproj`).
2. Rename the project file and directory.
3. Update `<Project>` and `<AssemblyName>` in the `.csproj`.
4. Rename namespaces and using statements in code files.
5. Update references in the solution and other projects.
